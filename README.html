📋 Project Overview

This is a comprehensive school website for CHEPTALAL GIRLS SECONDARY SCHOOL featuring:

  • Responsive Design - Works seamlessly on desktop, tablet, and mobile devices
  • School Colors - Blue (#003DA5), White, and Red (#E74C3C) throughout the design
  • Logo Management - Upload and display custom school logo
  • Photo Gallery - Upload and manage school photos and slides
  • Admin Panel - Easy-to-use dashboard for administrators
  • Multiple Pages - Home, About, Academics, Admissions, Contact
  • Forms & Storage - Contact forms, admission applications with localStorage

🚀 Getting Started

  1. Open the Website - Open index.html in a web browser
  2. Access Admin Panel - Click "Admin" in navigation or go to admin/admin.html
  3. Upload Logo - Use the admin panel to upload your school logo
  4. Upload Photos - Add photos to the gallery for the homepage slider
  5. Manage Content - Update school information, events, and other details

🎨 Customization

School Colors

The website uses three main colors:

  • Primary Blue: #003DA5
  • White: #FFFFFF
  • Accent Red: #E74C3C

These colors are defined in css/styles.css as CSS variables and can be easily customized.

School Information

Update school details in the admin panel:

  • School name and motto
  • Contact information
  • Location and address
  • School established year

Logo

Replace the placeholder logo via the admin panel. Recommended size: 500x500px (PNG, JPG, or GIF)

Photos

Add photos via the admin panel. They will appear in the homepage slider automatically.

🔧 Admin Panel Features

  • Dashboard - Overview of uploaded content and statistics
  • Logo Upload - Upload and preview school logo
  • Photo Gallery - Upload multiple photos with captions
  • School Information - Manage school details and description
  • Events Management - Add and manage school events
Note: All data is stored in the browser's localStorage. Data persists across page reloads but is specific to each browser/device.

💾 Data Storage

The website uses browser localStorage for data persistence:

  • schoolLogo - School logo image (base64)
  • galleryPhotos - Array of uploaded photos
  • schoolInfo - School information (name, contact, etc.)
  • schoolEvents - Array of school events
  • admissionApplications - Student admission applications
  • contactMessages - Contact form submissions

📱 Responsive Design

The website is fully responsive with breakpoints for:

  • Desktop - Full layout with all features
  • Tablet - (max-width: 768px) Optimized grid layouts
  • Mobile - (max-width: 480px) Single column, hamburger menu

🔐 Security Notes

This website uses client-side storage (localStorage). For production:

  • Implement backend server for data storage
  • Add user authentication for admin panel
  • Validate and sanitize all inputs
  • Use HTTPS for secure data transmission
  • Implement proper access controls
